Создайте программу для управления контактами, используя объектно-ориентированное программирование (ООП). Ваша программа должна включать следующие классы и функциональность:

Класс Contact:
Атрибуты:
    name (имя контакта)
    phone (номер телефона)
    email (электронная почта)
Методы:
    __str__() - метод, который возвращает строковое представление контакта в формате: "Имя: [name], Телефон: [phone], Email: [email]".

Класс ContactList:
Атрибуты:
    contacts (список контактов)
Методы:
    add_contact(contact) - метод для добавления контакта в список.
    remove_contact(name) - метод для удаления контакта по имени.
    find_contact(name) - метод для поиска контакта по имени. Если контакт найден, возвращайте объект контакта, иначе возвращайте сообщение о том, что контакт не найден.
    list_contacts() - метод для отображения всех контактов в списке.

Класс User:
Атрибуты:
    username (имя пользователя)
    contact_list (объект класса ContactList)
Методы:
    add_contact(name, phone, email) - метод, который создает новый контакт и добавляет его в контактный список.
    remove_contact(name) - метод, который удаляет контакт из контактного списка.
    show_contacts() - метод, который отображает все контакты пользователя.
